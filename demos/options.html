Available Options

  • displayMode

    Type:
    String
    Default:
    'blank'

    Mode of operation.

    Valid options: ['blank', 'button']

  • buttonMode

    Type:
    String
    Default:
    'button'

    Mode of button operation.

    Valid options: ['list', 'button']

  • Themeing

  • popupTheme

    Type:
    String
    Default:
    false
    Modes:
    all

    Theme for the popup.

  • popupOverlayTheme

    Type:
    String
    Default:
    false
    Modes:
    all

    Theme for the popup overlay.

  • buttonDefaultTheme

    Type:
    String
    Default:
    false
    Modes:
    button

    Default theme for the buttons in Button->Input mode, defaults to whatever themeDialog is set to.

  • padContent

    Type:
    Boolean
    Default:
    false
    Modes:
    all

    Add some padding in the dialog.

  • transition

    Type:
    String
    Default:
    'pop'
    Modes:
    all

    Animation style for transitions.

  • positionTo

    Type:
    String
    Default:
    'window'
    Modes:
    all

    Position popup in the middle of the 'window' or over any valid #selector.

  • Header Options

  • headerText

    Type:
    String
    Default:
    false
    Modes:
    all

    Text to use in the header. If this, and headerButton are FALSE, no header is used.

  • headerTheme

    Type:
    String
    Default:
    'a'
    Modes:
    button

    Theme for the header, if used.

  • headerMinWidth

    Type:
    String
    Default:
    false
    Modes:
    button

    Force Minimum Width of the dialog popup.

  • Display Options

  • dismissible

    Type:
    Boolean
    Default:
    true
    Modes:
    all

    Allow click outside to close (must use closeButton if false)

  • closeButton

    Type:
    String
    Default:
    false
    Modes:
    all

    Put a close button on the 'right' or 'left'

  • closeButtonTheme

    Type:
    String
    Default:
    'a'
    Modes:
    all

    Theme to use for that close button

  • Internal Operation

  • clickEvent

    Type:
    String
    Default:
    'click'
    Modes:
    all

    Event to bind all actions to (including header close button, buttons in button mode, and outside dialog trap).

  • Widget Callbacks

  • callbackOpen

    Type:
    Function
    Default:
    false
    Modes:
    all

    Function to run when the dialog opens.

  • callbackOpenArgs

    Type:
    Array
    Default:
    empty
    Modes:
    all

    An array of arguments to send to the open callback. Remember 'this' is the widget object.

  • callbackClose

    Type:
    Function
    Default:
    false
    Modes:
    all

    Function to run when the dialog closes, before cleanup is done - return false to prevent close.

  • callbackCloseArgs

    Type:
    Array
    Default:
    empty
    Modes:
    all

    An array of arguments to send to the close callback. Remember 'this' is the widget object.

  • Blank Mode Options

  • content

    Type:
    String
    Default:
    false
    Modes:
    blank

    A String of the HTML you wish to use in the blank/freeform mode. This may also be a jQuery object. Set to true if inlining options.

  • Button Mode Options

  • subTitle

    Type:
    String
    Default:
    false
    Modes:
    button

    A string to use as a prompt in the dialog, HTML allowed, wrapped in a <p>

  • button

    Type:
    Boolean
    Default:
    false
    Modes:
    button

    An object of buttons - see below for individual button options.

  • Button Object Options

  • id

    Type:
    String
    Default:
    false
    Modes:
    button/list

    The HTML ID of the button. If false, one is generated.

  • theme

    Type:
    String
    Default:
    inherited
    Modes:
    button

    The theme of the button.

  • icon

    Type:
    String
    Default:
    'check'
    Modes:
    button/list

    The icon of the button.

  • iconpos

    Type:
    String
    Default:
    'left'
    Modes:
    button

    The position of the icon on the button.

  • corners

    Type:
    Boolean
    Default:
    true
    Modes:
    button

    If true, round the corners of the button.

  • shadow

    Type:
    Boolean
    Default:
    true
    Modes:
    button

    If true, button will have a drop shadow.

  • close

    Type:
    Boolean
    Default:
    true
    Modes:
    button

    Clicking this button will close the dialog.

  • click

    Type:
    Function
    Modes:
    button

    A function to run when this button is clicked. If it returns false, the dialog will not close.

  • args

    Type:
    Array
    Modes:
    button

    An array of arguments to *also* pass to the click function. The first argument is always the event.
